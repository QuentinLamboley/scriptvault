# ScriptVault roadmap

This roadmap is directional rather than a promise of delivery dates.

## v0.2 — Research workflow depth

- Interactive local dashboard for branches, tags, snapshots, and file history.
- Project templates for sensitivity analyses, manuscript revisions, and preregistered workflows.
- Richer reproducibility records: package lockfiles, system information, and optional command provenance.
- Human-readable export of audit history for supplements and internal quality assurance.

## v0.3 — Exchange without cloud dependence

- Encrypted vault bundle export/import for portable backups and peer review.
- Read-only review mode with immutable milestone manifests.
- Optional signed integrity manifests for data-management plans and archived analyses.

## Longer term

- Additional IDE integrations while retaining the RStudio-first experience.
- Carefully scoped support for non-R scripts, notebooks, workflow managers, and manuscript assets.
- Community-defined conventions for reporting ScriptVault use in scientific methods and supplements.
